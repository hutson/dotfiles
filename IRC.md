# IRC

## Freenode

[Using the Network](https://freenode.net/using_the_network.shtml) is a guide on interacting with Freenode.

### User Mode

```
/mode hbetts +QRi
```

## Useful Commands

### Channel Modes

```
/mode [CHANNEL] +nt
```

### Query

Open a private "query" window with the designated user.

```
/query [NICK]
```

Close the current "query" window.

```
/query
```
